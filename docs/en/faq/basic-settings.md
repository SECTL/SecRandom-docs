# ::lucide:circle-help:: Basic Settings FAQ

### Q1: Why do some settings require restarting the software to take effect?

**A**: Some settings (such as language, DPI scaling, fonts, etc.) need to be loaded when the software starts, so the software needs to be restarted for them to take effect.

### Q2: What should I do if the startup with system setting fails?

**A**:
- Ensure the software is running with administrator privileges
- Check if system security software is blocking startup with system
- Try manually adding the software to system startup items

### Q3: What should I do if the software behaves abnormally after importing settings?

**A**:
- Try restoring default settings
- Re-import the previously backed up configuration file
- If the problem persists, please contact technical support

### Q4: What should I do if backup files take up too much space?

**A**:
- Reduce the automatic backup limit in backup management
- Delete unnecessary old backup files
- Uncheck unnecessary data types during manual backup

### Q5: What should I do if URL protocol registration fails?

**A**:
- Ensure the software is running with administrator privileges
- Check if other programs are using the same URL protocol
- Try manually registering the URL protocol
