# Quick Start

## What is it?

**[SecRandom](https://github.com/SECTL/Secrandom)** is a classroom random selection software designed specifically for educational scenarios, combining **fairness, ease of use, and aesthetics**:

- **Zero Learning Curve**: Easily complete random selection
- **Multi-scenario Coverage**: Single person/Multiple people/Group/Gender selection
- **Algorithm Guarantee**: Cryptographic-level random algorithm with verifiable fair results
- **Privacy Protection**: Local encrypted storage, data never exposed and protected from malicious tampering

## Applicable Scenarios

- **Education**: Classroom random roll call, random grouping, question lottery
- **Corporate Events**: Annual meeting lottery, team random grouping, employee benefit distribution
- **Competition Selection**: Random determination of contestant appearance order
- **Activity Organization**: Random lottery, lucky audience selection

## Core Features

### Selection Modes
- **Single Person Selection**: Randomly choose one student
- **Multiple Person Selection**: Select multiple students at once
- **Group Selection**: Extract by preset group rules
- **Gender Selection**: Precise selection by gender criteria

### Smart Settings
- **Dynamic Weight**: Intelligently adjust probability to ensure fairness
- **Probability Visualization**: Real-time update of selection probability
- **Animation Effects**: Smooth and exquisite selection animation experience

### Data Management
- **List Import**: Support xls/txt/csv format file import
- **Multi-class Management**: Easily manage multiple class lists
- **History Records**: Complete records with timestamps

### Privacy & Security
- **Local Storage**: Data never exposed
- **Password Protection**: Password verification for important operations

## Quick Start

1. **Download**: Get the latest version from [Download Page](/download)
2. **Install**: Extract to any non-system directory
3. **Launch**: Double-click `SecRandom.exe` to start the program
4. **Configure**: Click "Settings" → "List Management" → Import student list
5. **Use**: Return to main interface, choose selection mode, click "Start Selection"
