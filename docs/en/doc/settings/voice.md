> **Customize your voice broadcast experience**
>
> Through flexible voice settings, create the voice broadcast effect that best suits your usage habits. All settings are saved in real-time and take effect automatically.

## ::lucide:settings:: Basic Voice Settings

**Voice Function Switch**: Turn on or off voice broadcast function
- On: Enable voice broadcast function
- Off: Disable voice broadcast function

**Voice Engine**: Select voice synthesis engine type
- System TTS: Use system built-in voice synthesis engine
- Edge TTS: Use online voice synthesis service provided by Microsoft Edge

| Engine Type | Features | Requirements | Recommended Scenarios |
|-------------|----------|--------------|----------------------|
| **System TTS** | Offline use, fast response | Windows built-in | Daily teaching |
| **Edge TTS** | Good sound quality, multi-language | Requires internet connection | Public class, demonstration |

**Edge TTS Voice Role**: Select Edge TTS voice broadcast role
- Support multiple online voice roles

:::warning Edge TTS Notes
1. **Requires internet**: First use requires downloading voice package
2. **First time delay**: Each student's first draw will have voice generation delay
3. **Recommended operation**: First time use, complete a full round of drawing first, subsequent broadcasts will be smooth
4. **Speed setting**: After modifying speed, it is recommended to redraw a round
:::

**Speed Adjustment**: Adjust the speaking speed of voice broadcast
- The larger the value, the faster the speaking speed
- The smaller the value, the slower the speaking speed

**Broadcast Volume**: Adjust the volume size of voice broadcast
- The larger the value, the louder the volume
- The smaller the value, the softer the volume

**System Volume Control**: Whether to enable automatic system volume control
- On: Automatically adjust system volume to specified value when voice broadcasting
- Off: Keep system volume unchanged when voice broadcasting

**System Volume Size**: Set the target volume size when automatically controlling system volume
- Only takes effect when system volume control is enabled

## ::lucide:settings:: Specific Broadcast Settings

**Specific Broadcast Switch**: Turn on or off the general switch for specific result voice broadcast
- On: Enable voice broadcast function for specific results
- Off: Disable voice broadcast function for specific results

**Broadcast Mode**: Select voice broadcast mode
- Roll Call Mode: Modify specific broadcast content in roll call mode
- Lottery Mode: Modify specific broadcast content in lottery mode

**Select Class/Prize Pool**: Select the class or prize pool that needs TTS management
- Support selecting different classes or prize pools for separate settings

### Broadcast Content Settings

| Setting Item | Function Description |
|-------------|----------------------|
| **Switch** | Control whether to enable specific broadcast for this student/prize pool |
| **Student ID** | Display the student ID of this student/prize pool |
| **Name** | Display the name of this student/prize pool |
| **Replacement Name** | Enter the replacement name used for broadcasting |
| **Broadcast Prefix** | Set the prefix text for broadcast content |
| **Broadcast Suffix** | Set the suffix text for broadcast content |

:::tip
Voice settings recommendations:
- Choose clear and easy-to-understand voice roles to improve broadcast effect
- Adjust speaking speed according to students' acceptance ability
- If system volume changes frequently in class, it is recommended to enable system volume control function
:::
