# 快速开始

## 它是什么？

**[SecRandom](https://github.com/SECTL/Secrandom)** 是一款专为教育场景设计的班级抽号软件，集**公平性、易用性、美观性**于一体：

- **零门槛操作**：轻松完成随机抽取
- **多场景覆盖**：单人/多人/小组/性别抽取
- **算法保障**：密码学级随机算法，注重结果公平
- **隐私保护**：本地加密存储，数据不外泄且防恶意篡改

## 适用场景

- **教学领域**：课堂随机点名、随机分组、提问抽奖
- **企业活动**：年会抽奖、团队随机分组、员工福利分配
- **竞赛选拔**：参赛选手出场顺序随机决定
- **活动组织**：随机抽奖、幸运观众选取

## 核心特性

### 抽取模式
- **单人抽取**：随机选择一名学生
- **多人抽取**：一次抽取多名学生
- **小组抽取**：按预设小组规则抽取
- **性别抽取**：按性别条件精准抽取

### 智能设置
- **动态权重**：智能调节概率，确保公平
- **概率可视化**：实时更新选中概率
- **动画效果**：流畅和精美的抽取动画体验

### 数据管理
- **名单导入**：支持xls/txt/csv格式文件导入
- **多班管理**：轻松管理多个班级名单
- **历史记录**：带时间戳的完整记录

### 隐私安全
- **本地存储**：数据永不外泄
- **密码保护**：重要操作密码验证

## 快速开始

1. **下载**：从[下载页面](/download)获取最新版本
2. **安装**：解压至任意非系统目录
3. **启动**：双击 `SecRandom.exe` 启动程序
4. **配置**：点击"设置" → "名单管理" → 导入学生名单
5. **使用**：返回主界面，选择抽取模式，点击"开始抽取"
