# ::lucide:rocket:: 软件引导流程 

欢迎使用 SecRandom！首次启动软件时，引导向导将帮助您完成初始配置。本指南将详细介绍引导流程的每个步骤。

## ::lucide:clipboard-list:: 引导流程概览

软件引导包含以下 **9 个页面**，按顺序完成即可完成软件的初始配置：

| 步骤 | 页面 | 内容 | 预计耗时 |
|------|------|------|----------|
| 1 | [欢迎页](#1-欢迎页) | 显示应用Logo和标题 | 10 秒 |
| 2 | [语言选择页](#2-语言选择页) | 选择应用语言 | 30 秒 |
| 3 | [许可协议页](#3-许可协议页) | 同意许可协议和免责声明 | 1 分钟 |
| 4 | [数据迁移页](#4-数据迁移页) | 导入旧数据（可选） | 2 分钟 |
| 5 | [基本设置页](#5-基本设置页) | 简化模式、主题、主题颜色 | 1 分钟 |
| 6 | [名单管理页](#6-名单管理页) | 学生名单、班级管理 | 5 分钟 |
| 7 | [增强功能页](#7-增强功能页) | TTS、音乐、ClassIsland联动 | 3 分钟 |
| 8 | [测试页](#8-测试页) | 测试各项功能 | 2 分钟 |
| 9 | [链接页](#9-链接页) | 完成引导，查看相关链接 | 30 秒 |

::: tip 提示
完成整个引导流程大约需要 15-20 分钟。数据迁移和名单管理是可选步骤，您可以跳过稍后在设置中完成。
:::

---

## ::lucide:hand:: 1. 欢迎页

### ::lucide:layout:: 页面内容

- 显示 SecRandom 的 Logo 和标题
- 简洁的欢迎界面

### ::lucide:mouse-pointer-click:: 操作步骤

点击 **"开始"** 按钮进入下一步。

---

## ::lucide:languages:: 2. 语言选择页

### ::lucide:layout:: 页面内容

- 提供多种语言选项
- 选择后需要重启应用才能生效

### ::lucide:mouse-pointer-click:: 操作步骤

1. 从下拉列表中选择您偏好的语言（如"中文"、"English"）
2. 点击"下一步"继续

::: warning 注意
更改语言设置后需要重启应用才能生效。
:::

---

## ::lucide:scale:: 3. 许可协议页

### ::lucide:layout:: 页面内容

包含两个标签页：

- **许可协议**：显示 GPL 许可协议
- **免责声明**：显示软件免责声明

### ::lucide:mouse-pointer-click:: 操作步骤

1. **阅读协议**：在两个标签页之间切换，完整阅读协议内容
2. **滚动到底部**：必须滚动到协议底部才能勾选同意选项
3. **勾选同意**：
   - ::lucide:check:: 同意许可协议
   - ::lucide:check:: 同意免责声明
4. 点击"下一步"继续

::: warning 必须同意
必须同时勾选"同意许可协议"和"同意免责声明"才能继续下一步。
:::

---

## ::lucide:database:: 4. 数据迁移页

### ::lucide:layout:: 页面内容

- 提供手动导入旧数据的功能
- 如果您之前使用过 SecRandom，可以导入旧版本的数据

### ::lucide:mouse-pointer-click:: 操作步骤

#### 如果需要导入数据

1. 点击"导入"按钮
2. 选择旧版本的数据文件
3. 等待导入完成
4. 导入成功后会自动跳转到最后一页

#### 如果不需要导入数据

直接点击"下一步"跳过此步骤。

::: info 可选步骤
数据迁移是可选步骤，如果您是首次使用或不需要导入旧数据，可以跳过此步骤。
:::

---

## ::lucide:settings:: 5. 基本设置页

### ::lucide:layout:: 页面内容

配置软件的基本外观和行为设置。

### ::lucide:mouse-pointer-click:: 操作步骤

#### 1. 简化模式

- **开启**：隐藏高级功能，界面更简洁
- **关闭**：显示所有功能，适合高级用户
- **推荐**：首次使用建议关闭，以便了解所有功能

#### 2. 主题设置

选择界面主题：

| 选项 | 说明 |
|------|------|
| **浅色** | 浅色主题，适合白天使用 |
| **深色** | 深色主题，适合夜间使用，护眼 |
| **跟随系统** | 自动跟随系统主题设置 |

#### 3. 主题颜色

选择您喜欢的主题颜色，自定义软件外观。

::: tip 建议
建议选择"跟随系统"主题，这样软件会根据系统设置自动切换浅色/深色模式。
:::

## ::lucide:users:: 6. 名单管理页

### ::lucide:layout:: 页面内容

管理学生名单和班级信息。

### ::lucide:mouse-pointer-click:: 操作步骤

#### 1. 创建班级

1. 输入班级名称（如"高一(3)班"）
2. 点击"创建"按钮

#### 2. 导入学生名单

**方式一：手动输入**
- 按行输入学生姓名
- 可选：输入性别、小组信息

**方式二：Excel 批量导入（推荐）**

```
Excel 表格格式：
| 学号 | 姓名 | 性别 | 小组 |
|------|------|------|------|
| 01 | 张三 | 男   | A组  |
| 02 | 李四 | 女   | B组  |
```

**操作步骤**：
1. 准备 Excel 表格，确保第一行是列名
2. 点击"导入"按钮
3. 选择 Excel 文件
4. 系统自动识别列名并导入数据

#### 3. 创建奖池（可选）

1. 切换到抽奖模式
2. 输入奖池名称（如"期末奖品"）
3. 按行输入奖品名称
4. 设置奖品权重（权重越高，被抽中概率越大）

::: info 可选步骤
名单管理是可选步骤，您可以稍后在设置中继续配置。
:::

## ::lucide:box:: 7. 增强功能页

### ::lucide:layout:: 页面内容

包含三个标签页，配置软件的增强功能。

### ::lucide:mouse-pointer-click:: 操作步骤

#### 标签页 1：TTS 设置

配置语音合成功能：

| 设置项 | 说明 |
|--------|------|
| **语音引擎** | 选择 TTS 语音引擎 |
| **Edge TTS 语音名称** | 选择具体的语音（如"Microsoft Xiaoxiao Online"） |
| **音量大小** | 调整语音音量 |
| **语速** | 调整语音播放速度 |

#### 标签页 2：音乐设置

配置背景音乐：

| 设置项 | 说明 |
|--------|------|
| **动画音乐** | 抽取动画时播放的音乐 |
| **结果音乐** | 抽取结果展示时播放的音乐 |
| **音乐音量** | 调整音乐音量 |
| **淡入淡出** | 设置音乐的淡入淡出效果 |

#### 标签页 3：ClassIsland 联动

配置与 ClassIsland 的联动功能：

| 设置项 | 说明 |
|--------|------|
| **数据源设置** | 配置 ClassIsland 数据源 |
| **CSes 导入** | 导入 ClassIsland 的课程数据 |
| **通知服务** | 设置通知服务类型（0=关闭，1/2=启用） |
| **下课铃** | 配置下课铃功能 |

::: info 可选步骤
增强功能是可选步骤，您可以稍后在设置中继续配置。
:::

## ::lucide:test-tube-2:: 8. 测试页

### ::lucide:layout:: 页面内容

测试您配置的各项功能是否正常工作。

### ::lucide:mouse-pointer-click:: 操作步骤

#### 1. 测试 TTS 语音

- 点击"测试语音"按钮
- 听听语音播报是否正常
- 如果有问题，返回上一步调整设置

#### 2. 测试音乐播放

- 点击"测试动画音乐"按钮
- 点击"测试结果音乐"按钮
- 检查音乐是否正常播放
- 如果有问题，返回上一步调整设置

#### 3. 测试 ClassIsland 联动

- 点击"测试联动"按钮
- 检查与 ClassIsland 的连接是否正常
- 如果有问题，返回上一步调整设置

::: tip 建议
建议测试所有功能，确保配置正确后再继续。
:::
