## ::lucide:settings:: 系统要求检查

### 硬件要求
| 类别 | 最低 | 推荐 |
|------|------|------|
| **处理器** | 主频 1GHz 64位双核 | 4核心及以上 |
| **内存** | 200MB 可用内存 | 8GB+ 整机内存 |
| **存储** | 1GB 可用空间 | - |
| **显示器** | 1024×768 分辨率 | 推荐1080p |

### 软件环境
| 使用方式 | 系统要求 | Python 版本 |
|----------|----------|------------|
| **可执行文件** | Windows 10/11 & Linux | 无需安装 |
| **源码运行** | Windows 10/11 & Linux | Python 3.13 |

::: tip 
绝大部分 2017 年后出厂的教学一体机均满足最低或推荐配置要求。
:::

## ::lucide:zap:: 快速开始

### ::lucide:download:: 1. 下载安装

从[下载页面](https://stk.sectl.top/SecRandom)获取最新版本，进入下载界面的文件列表后，选择合适的文件版本：
- 一般情况下，最新版本位于文件列表的最顶部
- 如果需要下载旧版本，可以在文件列表中找到对应的版本号
- 带有 beta 后缀的版本为预览版本，后缀之后的数字表示预览版本的迭代次数
- 带有 alpha 后缀的版本为开发版本，后缀之后的数字表示开发版本的迭代次数  
点击需要的版本的文件夹，进入文件选择界面，选择合适的文件：
- 名称中带有`Windows-Setup`的文件为 Windows 安装版本
- 名称中带有`Windows-Portable`的文件为 Windows 便携版本
- 名称中带有`Linux`的文件为 Linux 版本  
点击需要的文件进入下载界面，点击“下载”按钮，即可开始下载。
:::tip
建议使用多线程下载工具以提升下载速度。
:::

文件下载完成后：

- **Windows 用户**：
  安装版：下载完成后，打开下载的可执行文件，按照提示进行安装。
  便携版：将压缩包解压到您想让软件长久停留的地方。

   ::: info 如何找到下载的文件
   - 对于大部分浏览器，下载的文件位于"下载"文件夹中。可以在文件资源管理器中打开"下载"文件夹，找到下载的文件。
   - 部分浏览器可能在下载开始时询问保存位置，下载完成后在选择的保存位置寻找即可。
   :::

   ::: info 如何判断系统版本及处理器架构
   - 右键点击“此电脑”或“计算机”，点击“属性”；
   - 在“设备规格”中找到系统类型；
   - 如果显示“64 位操作系统”，则为 64 位系统，如果显示“32 位操作系统”，则为 32 位系统。
   :::

   ::: info 如何解压压缩包
   - 找到你下载的压缩包文件，鼠标右键点击；
   - 在出现的右键菜单中点击“全部解压缩”；
   - 选择解压的文件保存位置（默认保存到相同目录下的新文件夹中），点击“提取”。
   :::
   
- **Linux 用户**：下载 .deb 安装包

### ::lucide:rocket:: 2. 启动程序

- **Windows**：双击 `SecRandom.exe` 启动程序
- **Linux**：通过图形界面自带的应用菜单或命令行启动

如果您是初次启动，会进入引导界面。  
引导界面内容介绍可参考[这个页面](software-guide.html)

### ::lucide:dices:: 3. 开始抽取

1. 返回主界面
2. 选择抽取模式（点名/抽奖）
3. 设置抽取参数
4. 点击"开始抽取"按钮
5. 查看抽取结果
