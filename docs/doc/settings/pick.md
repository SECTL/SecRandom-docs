## ::lucide:settings:: 一、抽取设置

抽取设置包含四个独立选项卡：**点名设置**、**闪抽设置**、**抽奖设置**和**人脸抽取**。前三个设置都包含抽取功能、显示设置、动画设置三个部分。**人脸抽取**选项卡用于配置人脸抽取功能。

### ::lucide:library:: 1.点名设置

#### ::lucide:settings:: 1.1 独立名单配置

**功能说明**：为每个名单应用特殊设置，包括抽取功能、显示设置、动画设置等。

**操作方式**：
1. 点击设置项右侧的"打开"按钮，弹出独立名单配置界面
2. 在"选择名单"下拉框中选择要配置的名单
3. 参考下方的设置项介绍配置需要的设置，所有设置将实时应用
4. 如果不再需要该名单的特殊设置，点击"选择名单"板块的"同步主设置"按钮，即可跟随全局设置。

### ::lucide:lightbulb:: 4. 配置建议

#### ::lucide:dices:: 4.1 点名设置建议

- **抽取模式**：根据教学需求选择
  - 重复抽取：适合需要同一人被多次点名的场景（bushi）
  - 不重复抽取：适合确保每个学生都被点到
  - 半重复抽取：平衡重复和不重复，可设置重复次数
- **动画间隔**：建议 80-100 ms，既流畅又不太快
- **字体大小**：建议 50-80 px，根据显示屏幕大小调整
- **学生图片**：如需个性化展示可启用

#### ::lucide:zap:: 4.2 闪抽设置建议

- **抽取人数**：建议 1-3 人，避免一次抽取过多
- **点击后禁用时间**：建议 3-5 秒，防止误操作
- **动画间隔**：建议 80-100 ms
- **字体大小**：建议 50-80 px

#### ::lucide:gift:: 4.3 抽奖设置建议

- **抽取模式**：根据实际情况选择，确保奖品公平分配
- **奖品图片**：建议启用，增加视觉效果
- **动画间隔**：建议 80-100 ms
- **字体大小**：建议 50-80 px

### ::lucide:box:: 5. 特殊功能说明

1. **文件监视器**：点名设置和抽奖设置会自动监视名单/奖池文件夹的变化，实时更新下拉框选项

2. **抽取模式联动**：
   - 重复抽取模式：禁用清除记录和半重复次数设置
   - 不重复抽取模式：启用清除记录，禁用半重复次数
   - 半重复抽取模式：启用清除记录和半重复次数设置

3. **后台加载**：闪抽设置和抽奖设置使用后台线程加载数据，避免界面卡顿

## ::lucide:app-window:: 二、浮窗管理

### ::lucide:settings:: 1. 基本设置

**启动时显示浮窗**：控制软件启动时是否自动显示浮窗

**浮窗透明度**：调整浮窗透明度
- 范围：0%-100%

**置顶模式**：选择浮窗置顶方式
- 关闭置顶：浮窗不置顶，与其他窗口层级相同
- 置顶：浮窗以普通权限置顶显示在所有窗口上方
- UIA置顶：浮窗使用 UIAccess 层级置顶显示在所有窗口上方（可实现显示在 Windows 登录界面或全屏 UWP 应用上方）

**浮窗可拖动**：控制浮窗是否可被拖动
- 启用：允许拖动浮窗到任意位置
- 禁用：浮窗固定在当前位置不可拖动

**长按时间**：设置拖动浮窗前需要长按浮窗的时间

**无焦点模式**：设置浮窗是否可获取应用焦点
- 启用：浮窗不可获取应用焦点
- 禁用：浮窗可获取应用焦点，可能导致前台窗口变化时浮窗获得焦点

**扩展闪抽组件**：在浮窗闪抽按钮旁显示更多选择控件
- 启用：在闪抽按钮旁额外显示一个下箭头，用于设置抽取范围、性别等
- 禁用：仅显示闪抽按钮

**重置浮窗位置**：将浮窗位置重置为默认位置（一般位于屏幕的左上角）

### ::lucide:settings:: 2. 外观设置

**浮窗控件配置**：选择在浮窗中显示的功能按钮
- 点击设置项最右边的下箭头即可展开按钮设置下拉框，勾选需要的按钮即可让其在浮窗中显示
- 在下箭头的左边最多可显示其中 2 个按钮，点击按钮名称旁边的 × 即可快捷移除该按钮

**浮窗排列方式**：设置浮窗中控件排列方式
- 横向排列：控件水平方向排列
- 纵向排列：控件垂直方向排列
- 矩形排列：控件以矩形布局排列

**浮窗显示样式**：设置浮窗中控件显示样式
- 图标+文字：同时显示功能图标和文字说明
- 仅图标：只显示功能图标
- 仅文字：只显示功能文字

**浮窗大小**：设置浮窗按钮和图标的大小  
注：浮窗大小均为 1920×1080 100% 缩放的数据，不同缩放比例下可能会存在差异

| 超级小 | 20px×20px          | 20px×20px            |
| 超小   | 30px×30px          | 25px×25px            |
| 小     | 40px×40px          | 28px×28px            |
| 中     | 50px×50px          | 30px×30px            |
| 大     | 60px×60px          | 35px×35px            |
| 超大   | 70px×70px          | 40px×40px            |

### ::lucide:settings:: 3. 贴边设置

**贴边功能**：控制浮窗靠近屏幕边缘时是否自动收纳为悬浮按钮

**贴边收纳时间**：设置浮窗移动到边缘到收纳的时间间隔

**贴边显示样式**：设置浮窗收纳为悬浮按钮时的显示样式
- 文字：显示文字提示
- 图标：显示功能图标
- 箭头：显示箭头

### 前台隐藏设置

**功能说明**：配置当特定窗口处于前台（活跃）状态时，自动隐藏 SecRandom 的悬浮窗，避免遮挡重要内容。

#### 1. 前台窗口隐藏浮窗

**功能说明**：总开关，控制是否启用“前台窗口检测并隐藏浮窗”的功能。

- 开启：当检测到满足条件的窗口处于前台时，浮窗会自动隐藏。
- 关闭：浮窗始终保持显示（除非手动隐藏），不会因前台窗口变化而自动隐藏。

#### 2. 前台窗口标题

**功能说明**：根据窗口标题关键词进行匹配。当处于前台的窗口标题包含设定关键词时，浮窗隐藏。

**操作方式**：
- 输入英文分号 `;` 分隔的关键词列表。
- 示例：`PowerPoint;Word;Excel`
- 逻辑：只要前台窗口的标题中**包含**任意一个关键词，即触发隐藏。

#### 3. 前台进程名

**功能说明**：根据进程名称进行匹配。当处于前台的窗口属于设定进程时，浮窗隐藏。

**操作方式**：
- 输入英文分号 `;` 分隔的进程名列表（通常以 .exe 结尾）。
- 示例：`WeChat.exe`
- 逻辑：只要前台窗口所属的进程名与列表中的任意一项**完全匹配**，即触发隐藏。

## ::lucide:settings:: 三、个性设置小贴士

### 1. 抽取设置优化

- **显示效果**：根据教室大小调整字体大小，确保后排学生也能清晰看到
- **动画体验**：适当调整动画间隔和播放次数，平衡视觉效果和抽取效率
- **个性化主题**：开启颜色主题或设置艺术字体，让抽取过程更加生动

### 2. 浮窗设置技巧

- **便捷操作**：按钮数量设置为“抽人+闪抽”，提升课堂抽取效率
- **界面整洁**：根据实际情况选择浮窗控件和显示样式，减少视觉干扰
- **智能贴边**：利用贴边功能，既保持浮窗便捷性，又不影响其他软件使用
- **前台隐藏**：根据需要配置隐藏条件，避免遮挡重要内容
