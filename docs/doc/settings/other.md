> ⚙️ **其他功能设置**
> 
> 通过灵活的快捷键和更新设置，提升您的使用效率，所有设置实时保存并自动生效。

## 🎯 快捷键设置

**启用快捷键**：启用后可使用快捷键快速操作
:::tip
快捷键均为全局快捷键，无论在哪个页面都可触发
:::
- 开启：启用快捷键功能
- 关闭：禁用快捷键功能

### ⌨️ 快捷键设置

设置打开功能的快捷键

:::warning
- 快捷键设置应避免与其他软件冲突，以免导致意外操作
:::

## 🎯 更新设置

**强制检查更新**：强制将应用更新到当前通道上的最新版本，即使此版本比应用当前版本更旧。

**自动检查更新并安装**：设置应用的自动更新模式
- 从不自动更新：完全不自动检查更新
- 自动检查更新并通知：自动检查更新，并在有新版本时通知用户
- 自动检查更新并下载：自动检查更新并下载，但不自动安装
- 自动下载更新并安装：自动检查、下载并安装更新

**更新通道选择**：选择软件更新发布通道
- 稳定通道：获取经过充分测试的稳定版本
- 发布预览通道：提前预览下一个大版本中应用的功能，包含较新的特性和改进，可能存在少量缺陷。
- 测试通道：接收应用最新的测试版更新，包含最新的特性和改进，可能包含缺陷和末完工的功能。


**更新源选择**：选择软件更新下载来源
- 自动检测延迟：自动检测更新源延迟，选择最快的更新源下载更新
- github：从GitHub下载更新，国内访问速度较慢
- ghfast/ghproxy：从ghfast/ghproxy下载更新，速度更快

:::tip
其他设置建议：
1. 根据个人使用习惯设置常用功能的快捷键
2. 定期检查软件更新，获取最新功能和安全修复
3. 根据稳定性需求选择合适的更新通道
4. 选择网络条件较好的更新源，提高更新速度
:::
