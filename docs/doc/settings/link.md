> 🔗 **设置课表联动功能**
> 
> 通过灵活的课表联动配置，实现课堂与下课的专属功能，所有设置实时保存并自动生效。

## 🎯 数据源选择

**数据源选择**：选择课程表数据源
- 不使用数据联动：完全独立使用本系统
- 使用CSES课表数据：使用来自CSES课表的课程表数据
- 使用Classisland数据源：使用来自Classisland的课程表数据（需要在Classisland中安装联动插件）

## 🎯 CSES课程表导入

**课程表导入**：导入CSES课程表
- 从文件导入：通过文件导入CSES课程表
- 查看当前配置：查看已有的CSES课程表配置

## 🎯 课间禁用设置

**课间禁用功能**：在下课时间段内抽取需要安全验证
- 开启：在下课时间段内抽取需要进行安全验证
- 关闭：下课时间段内抽取无需安全验证

**验证流程功能**：在非上课时间段触发时将弹出安全验证；若关闭则直接禁用组件
- 开启：非上课时间触发需要安全验证
- 关闭：非上课时间直接禁用相关组件（不隐藏组件）

**上课前提前解锁时间**：设置上课前自动解锁的时间
- 数值越大，解锁时间越早；数值越小，解锁时间越晚
- 设置为0表示准时解锁

## 🎯 课前重置设置

**课前重置功能**：在上课前指定秒数内自动清除临时记录和界面结果
- 开启：在上课前指定秒数自动清除临时记录和界面结果
- 关闭：不进行自动清除

**课前重置时间**：设置上课前自动清除临时记录和界面结果的时间（1-1440秒）
- 数值越大，清除时间越早
- 数值越小，清除时间越晚
- 设置为0表示准时清除

## 🎯 科目历史记录过滤

**科目历史记录过滤**：计算权重时只使用当前科目的历史记录
- 开启：仅使用当前科目课程时间中产生的历史记录计算权重
- 关闭：使用全部历史记录计算权重

:::tip
联动设置建议：
1. 根据实际需求选择合适的数据联动方式
2. 定期导入最新的课程表数据以确保准确性
3. 合理设置课间禁用和课前重置功能，提高课堂管理效率
4. 根据教学需要配置科目历史记录过滤功能
:::

## 🌐 ClassIsland通知服务

**通知服务类型**：选择通知服务类型
- SecRandom：使用 SecRandom 通知窗口发送通知
- ClassIsland：通过联动插件使用 ClassIsland 发送通知
- SecRandom + ClassIsland：同时使用 SecRandom 和 ClassIsland 发送通知

:::warning
使用 ClassIsland 服务前，需要先在 ClassIsland 中安装 ClassIsland 插件和 NET.8 运行时。
:::

**通知显示时长**：设置 ClassIsland 通知显示时长（秒）
- 控制 ClassIsland 通知的显示时间，确保您有足够时间查看通知内容
