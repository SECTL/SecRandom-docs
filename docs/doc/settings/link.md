> ::lucide:link:: **设置课表联动功能**
> 
> 通过灵活的课表联动配置，实现课堂与下课的专属功能，所有设置实时保存并自动生效。

## ::lucide:target:: 数据源选择

**数据源选择**：选择课程表数据源
- 不使用数据联动：完全独立使用本系统
- 使用CSES课表数据：使用来自CSES课表的课程表数据
- 使用Classisland数据源：使用来自Classisland的课程表数据（需要在Classisland中安装联动插件）

## ::lucide:target:: CSES课程表导入

**课程表导入**：导入CSES课程表
- 从文件导入：通过文件导入CSES课程表
- 查看当前配置：查看已有的CSES课程表配置
- 清除课表：清除已有的CSES课程表配置

## ::lucide:target:: 课间禁用设置

**课间禁用功能**：在下课时间段内抽取需要安全验证
- 开启：在下课时间段内抽取需要进行安全验证
- 关闭：下课时间段内抽取无需安全验证

**验证流程功能**：在非上课时间段触发时将弹出安全验证；若关闭则直接禁用组件
- 开启：非上课时间触发需要安全验证
- 关闭：非上课时间直接禁用相关组件（不隐藏组件）

**上课前提前解锁时间**：设置上课前自动解锁的时间
- 数值越大，解锁时间越早；数值越小，解锁时间越晚
- 设置为0表示准时解锁

**下课后延迟禁用时间**：设置下课后延迟触发课间禁用的时间
- 用于给老师留出拖堂或下课缓冲时间
- 数值越大，延迟禁用的时间越长

**下课时隐藏浮窗**：当检测到下课（非上课时段）时自动隐藏浮窗
- 开启：下课自动隐藏，上课自动显示（若符合显示条件）
- 关闭：浮窗显示不受课程状态影响

## ::lucide:target:: 课前重置设置

**课前重置功能**：在上课前指定秒数内自动清除临时记录和界面结果
- 开启：在上课前指定秒数自动清除临时记录和界面结果
- 关闭：不进行自动清除

**课前重置时间**：设置上课前自动清除临时记录和界面结果的时间（1-1440秒）
- 数值越大，清除时间越早
- 数值越小，清除时间越晚
- 设置为0表示准时清除

## ::lucide:target:: 科目历史记录过滤

**科目历史记录过滤**：计算权重时只使用当前科目的历史记录
- 开启：仅使用当前科目课程时间中产生的历史记录计算权重
- 关闭：使用全部历史记录计算权重

**课间归属**：课间时段的科目历史记录归属到哪一类
- 上节课：课间时段的科目历史记录归属到上节课的科目
- 下节课：课间时段的科目历史记录归属到下节课的科目
- 课间：课间时段的科目历史记录归属到单独的课间时段

:::tip 联动设置建议
1. 根据实际需求选择合适的数据联动方式
2. 定期导入最新的课程表数据以确保准确性
3. 合理设置课间禁用和课前重置功能，提高课堂管理效率
4. 根据教学需要配置科目历史记录过滤功能
:::
