## 🎯 抽取设置

### 🔄 抽取模式
**重复抽取**
- ✅ **允许**：学生可多次被抽中
- **半重复**
- ❌ **禁止**：一轮内每个学生只抽一次

**适用场景**：
- 允许重复 → 日常点名、活跃气氛
- 禁止重复 → 考试抽题、确保覆盖

### ⚖️ 抽取方式
**公平抽取算法**
- 🎯 **启用**：基于权重算法，确保抽取机会均等
- 📊 **禁用**：纯随机抽取，概率均等

**算法优势**：动态调整权重，长期更公平


### 🎨 显示与动画设置
| 设置项 | 选项 | 说明 |
|--------|------|------|
| **字体大小** | 10-2000px | 适配不同显示器 |
| **显示格式** | 姓名/学号/姓名+学号 | 多种显示方式 |
| **动画效果** | 纯色/颜色变换 | 丰富的视觉效果 |
| **动画速度** | 慢/中/快 | 可调节动画节奏 |

**课表软件联动**：现已支持使用ci/cw展示抽人结果（仅支持闪抽），在点名设置-闪抽设置中打开`是否使用CI/CW显示结果`即可

### 🎵 背景音乐设置
**音频支持**
- 🎼 支持格式：MP3、WAV、FLAC
- 📁 自定义上传：支持拖放添加
- 🔊 独立音量：0-100%精确调节
- 🎚️ 淡入淡出：平滑音频过渡

**使用场景**：
	- 🎶 抽取过程播放音乐，提升专注度
	- 📢 公布抽取结果时播放提示音效
	- 🏫 营造轻松愉快的课堂氛围

## 🔊 语音播报设置

### 🗣️ 语音引擎选择
| 引擎类型 | 特点 | 要求 | 推荐场景 |
|----------|------|------|----------|
| **系统TTS** | 离线使用、响应快 | 仅 Windows 版本支持 | 日常教学 |
| **Edge TTS** | 音质好、多语言 | 需要联网 | 日常教学/公开课、展示 |

::: warning Edge TTS 注意事项
1. 🌐 **需要联网**：首次使用需下载语音包  
2. ⏱️ **首次延迟**：每个学生首次抽取会有语音生成延迟  
3. 🔄 **建议操作**：首次使用时先完整抽取一轮，后续即可流畅播报  
4. 🎵 **语速设置**：修改语速后建议重新抽取一轮
:::

### 🎛️ 播报效果控制
**音量控制**
- 🔊 **播报音量**：独立于系统音量
- 🎚️ **音量恢复**：播报后自动恢复原始音量

**语速调节**
- 🐌 **慢速**：适合听力训练
- 🏃 **中速**：日常教学使用
- ⚡ **快速**：节省时间场景

## 🔒 安全设置

**配置保护**
- 🔐 **设置密码**：为设置界面添加密码锁
- 🎭 **防止篡改**：避免学生恶意修改名单

**使用控制**
- ⏰**课间禁用**：防止学生课间乱抽人影响使用
- 🔒**使用授权**：使用软件功能前需要进行验证
- ⏰**时间锁**：可设置特定时段内锁定

**安全提示**
- 🔄 定期更换密码
- 📱 支持密码找回功能
- 🚫 连续错误锁定机制

## 💡 设置使用小贴士

**日常教学场景**
