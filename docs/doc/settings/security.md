> **保障您的配置安全**
> 
> 通过灵活的安全配置，保护您的系统免受未授权访问，所有设置实时保存并自动生效。

## ::lucide:lock:: 验证方式

**安全开关**：启用后所有安全操作都需要验证密码
- 开启：选定的安全操作均需密码验证
- 关闭：进入设置等安全操作无需密码验证

**设置/修改密码**：设置或修改安全验证密码
- 点击按钮进入密码设置界面
- 建议使用强密码，包含字母、数字和特殊字符

**TOTP验证**：启用后可在安全操作中使用TOTP验证
- 开启：支持使用TOTP作为验证方式
- 关闭：不使用TOTP验证功能

**设置TOTP**：配置TOTP动态口令验证（即2FA验证）
- 点击按钮进入TOTP密钥设置界面

**U盘验证**：启用后可在安全操作中使用U盘验证
- 开启：支持使用U盘作为验证方式
- 关闭：不使用U盘验证功能

**绑定U盘**：绑定用于验证的U盘设备
- 插入U盘后点击按钮完成绑定
- 绑定后该U盘可作为验证凭证

**解绑U盘**：解除U盘设备绑定
- 点击按钮解除已绑定的U盘
- 解绑后该U盘不再作为验证凭证

## ::lucide:footprints:: 安全验证步骤

**安全验证步骤**：选择安全验证组合方式
- 单步验证（任选一种方式）：从可用验证方式中选择一种完成即可完成验证
- 仅密码/仅TOTP/仅U盘：仅需进行选定的验证方式即可完成验证
- 多步验证（组合验证方式）：需要完成多种验证方式

## ::lucide:mouse-pointer-click:: 安全操作

**设置预览开关**：进入设置验证时允许预览设置
:::tip
要使用预览功能，必须先开启**安全开关**和**打开设置验证**安全操作。
:::
- 开启：验证前可以预览设置内容，进行验证后才能修改设置
- 关闭：不启用预览功能

**显示/隐藏浮窗验证**：启用后显示或隐藏浮窗时需要安全验证
- 开启：浮窗的显示/隐藏操作需要验证
- 关闭：浮窗的显示/隐藏操作无需验证

**重启/退出验证**：启用后退出软件时需要安全验证
- 开启：重启/退出软件时需要进行安全验证
- 关闭：重启/退出软件时无需验证

**打开设置验证**：启用后打开设置窗口时需要安全验证
- 开启：打开设置窗口时需要进行安全验证
- 关闭：打开设置窗口时无需验证

:::tip 安全设置建议
1. 启用安全开关增强系统安全性
2. 设置强密码并定期更换
3. 根据需要启用U盘/TOTP验证作为双重保障
4. 对重要操作（如退出软件、打开设置）启用验证
:::
