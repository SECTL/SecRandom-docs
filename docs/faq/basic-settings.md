# ❓ 基础设置常见问题

### Q1: 为什么某些设置需要重启软件才能生效？

**A**: 部分设置（如语言、DPI缩放、字体等）需要在软件启动时加载，因此需要重启软件才能生效。

### Q2: 开机自启设置失败怎么办？

**A**:
- 确保以管理员权限运行软件
- 检查系统安全软件是否阻止了开机自启
- 尝试手动将软件添加到系统启动项

### Q3: 导入设置后软件出现异常怎么办？

**A**:
- 尝试恢复默认设置
- 重新导入之前备份的配置文件
- 如果问题持续，请联系技术支持

### Q4: 备份文件占用空间过大怎么办？

**A**:
- 在备份管理中减少自动备份上限
- 删除不需要的旧备份文件
- 在手动备份时取消不需要的数据类型

### Q5: URL协议注册失败怎么办？

**A**:
- 确保以管理员权限运行软件
- 检查是否有其他程序占用了相同的URL协议
- 尝试手动注册URL协议
