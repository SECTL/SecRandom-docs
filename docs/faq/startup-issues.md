# <img src="/fluent-icons/help-circle.svg" alt="help-circle" width="20" height="20" style="vertical-align: middle; margin-bottom: 2px;" /> 程序无法启动          

## **症状1**：修改设置后程序无法启动

**解决方案**：
- 1. 关闭程序（检查系统托盘或任务管理器确保完全退出）
- 2. 删除全部配置文件：`程序目录/app/Settings` 文件夹
- 3. 重新启动程序
- 4. 在托盘图标→设置中重新配置
- 5. 如果你确信这是一个 bug ，可将出现问题的配置文件反馈给开发者以便定位问题。

## **症状2**：程序卡在"注册 URL 协议"后闪退（尤其在1.3.x.x版本中）                                                
**解决方案**：
- 1. 如果是初次启动遇见此问题，先尝试重启电脑，看SecRandom是否自启动，如果自启动成功，以后就不会出现此问题
- 2. 如未自启动（可能弹出DLL报错窗口），尝试[点击此处](https://aka.ms/vc14/vc_redist.x64.exe)下载并安装VC运行库
- 2. 重启电脑
- 3. 重新启动程序
- 4. 如仍存在问题，请反馈至开发者
