## 🎯 界面概览

SecRandom采用现代化的界面设计理念，通过清晰的层次结构和直观的操作流程，让每位用户都能快速上手。界面支持**智能适配**和**个性化定制**，完美适配不同使用场景。

## 🎨 主题与个性化

### 🌈 **预设主题方案**

#### ☀️ **浅色主题**
- **适用场景**：教室、办公室等明亮环境
- **配色特点**：白底黑字
- **护眼指数**：⭐⭐⭐
- **能耗优化**：⭐⭐⭐⭐

#### 🌙 **深色主题**
- **适用场景**：晚间使用、昏暗环境
- **配色特点**：黑底白字
- **护眼指数**：⭐⭐⭐⭐⭐
- **专注模式**：减少眼部疲劳
